# Olá, pessoal! 

Criei este repositório para compartilhar meus estudos e ajudar quem deseja praticar a Linguagem C. O conteúdo reúne exercícios autorais e desafios propostos durante o curso de Análise e Desenvolvimento de Sistemas na UFPR.

Atualmente, tem mais de 112 exercícios, todas as resoluções foram feitas por mim sem nenhum uso de IA, fiquem a vontade para treinarem, comentar ou sugerir melhores soluções se quiserem. Os níveis de exercícios são graduais, desde o mais básico até o avançado
